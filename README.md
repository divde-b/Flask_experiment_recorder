# 项目名称 

基于python，flask，mysql的网络试验记录系统。

## 功能特性  

-✅功能1： 实现数据的基本增删改查。  
-✅功能2： 查看网络报告的详情，使用markdown。  
-✅功能3： 文本的编辑功能。  
-✅功能4： 搜索实验报告记录，支持模糊搜索。  
-⏳功能5： 用户登录。  
-⏳功能6： 构建AI系统到项目网页中。  

## 技术栈  

-后端： python + Flask  
-数据库： MySQL  
-前端： HTML/CSS/JavaScript  
-其它： Markdown
